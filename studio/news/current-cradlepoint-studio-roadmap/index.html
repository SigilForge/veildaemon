Print

Future print editions will integrate with the same ownership system used for digital recoveries, allowing authenticated operators to order current Archive editions directly from their library.

Audio

Planned audiobook releases and narrated Archive recoveries will become part of the same connected library.

The Archive Client

The long-term vision is a dedicated Archive terminal.

Not simply a storefront.

An application where Operators can:

  • Browse owned recoveries
  • Read books
  • Listen to audio
  • Download updated editions
  • Purchase new Archive materials
  • Access dossiers
  • Receive recovered transmissions
  • Track Archive synchronization events

Every new release strengthens the same ecosystem.

The goal has never been to build a collection of disconnected products.

The goal is to build a universe people can step into.
