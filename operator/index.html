The Handler sets the target number and tells you what success, failure, or misfire means.

Quick read: 10 = easy under pressure // 13 = risky / standard // 16 = hard // 19 = severe // 22 = nearly impossible

Core rule: Misfire is margin-based.

  • Total ≥ TN + 3: Clean success
  • Total ≥ TN: Success
  • Total is TN − 1 or TN − 2: Misfire / wrong-success
  • Total ≤ TN − 3: Failure
  • Natural 3: Hard failure or severe misfire, Handler choice

Misfire means: The Operator gets part or all of what they wanted, but the scene changes against them.

Example (TN 13): 16+ = Clean success // 13–15 = Success // 11–12 = Misfire // 10 or less = Failure // Natural 3 = Hard failure / severe misfire
