# Acceptance rubric

CA-001 passes only when all five direct-engine runs and all five real UI runs return readable structured local output, preserve both ownership/control and trust/extraction, resolve as standalone thoughts, fit platform limits, and meet character validation thresholds without warnings.

Static checks bind the artifact to current Relay sources, architecture labels, production project, and declared call limits. This is a bounded semantic regression test, not a proof of general comprehension.
